Draws the shape on the screen. This is a pure virtual function that must be implemented by all derived classes.

Parameters
configThe configuration settings for drawing, including line color, thickness, and bias.
  • line_color defines the color of the shape's outline.
  • line_thickness determines how thick the outline will be.
  • bias is used to adjust the shape's position on the screen.
