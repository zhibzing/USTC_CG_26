# 3D框架配置说明

首先切换工作目录到作业仓库根目录下，运行git命令初始化子模块：

```shell
git submodule update --init --recursive --remote
```

3D作业框架会被放在Framework3D/Ruzino文件夹下，将工作目录切换到Ruzino文件夹中，阅读README[Ruzino-Homework/README.md at main · SyouSanGin/Ruzino-Homework](https://github.com/SyouSanGin/Ruzino-Homework/blob/main/README.md)，进行3D框架配置
